# 男频故事引擎

本文件在 [编剧共同内核](screenwriting-core.md) 之上提供男频版的选题、进展系统、对手层级、团队关系和质量门槛。先执行共同内核，再执行本文件。

“男频”是发行与叙事惯例的市场分类，不是对所有男性的心理诊断。不要假设男性天然只喜欢暴力、财富、后宫、权力或让所有人臣服。根据具体项目选择能力成长、问题解决、资源跃迁、团队、责任、探索或秩序等观看承诺。

爱情线权重可以为 0。出现女性角色时，必须为她们保留独立目标、能力、边界和选择，不把女性写成奖品、数量指标或无条件崇拜者。

本引擎提供创作判断，不承诺爆款、播放量、付费率或平台推荐。

本文的“当前阶段”在开启分季时指当前季，在未分季时指全剧或用户指定的当前规划范围。

## 一、男频分支的执行原则

1. **让进展可见。** 每次胜利至少改变技能、资源、声望、团队、权限、信息、组织范围或责任中的一项。
2. **让胜利来自行动。** 身份、系统、血脉和贵人只能提供条件，关键任务必须由主角判断、执行并承担代价。
3. **让问题真实变难。** 对手会学习，规则会收紧，责任会扩大；不要用同一种纨绔换皮制造升级。
4. **让能力受规则约束。** 强能力需要边界、成本、反制或责任，否则进展没有重量。
5. **让团队拥有功能和主体性。** 队友不是喊口号的小弟，导师不是代打，女性不是奖励。
6. **让地位服务主题。** 财富、权力、修为和组织规模是故事资产，不是人物价值的唯一证明。
7. **让责任通过选择体现。** 家庭、团队、社区或家国责任必须落在具体风险、牺牲和行动上，不靠口号。
8. **同时保护逻辑与情绪。** 热血、爽感和身份反差必须建立在因果、证据和真实后果上。

## 二、观看承诺系统

每个项目选择一个主要承诺和至多两个次要承诺。不要同时追求所有升级维度。

| 承诺 | 观众获得什么 | 可见兑现 |
|---|---|---|
| 能力掌握 | 看见人物从不会到会、从会到精 | 完成训练、解决难题、形成独特方法 |
| 问题解决 | 看见复杂局面被判断、拆解和执行 | 案件突破、工程完成、危机解除、策略生效 |
| 资源与地位跃迁 | 看见人物拥有更大行动空间 | 获得资金、权限、声望、领地或组织位置 |
| 团队与事业建设 | 看见个人能力转化为可持续组织 | 招募互补成员、建立流程、完成共同任务 |
| 公平与秩序重建 | 看见主角理解规则并修正不公 | 揭露漏洞、建立新制度、保护弱者、追责 |
| 探索与世界解锁 | 看见未知规则、地图或文明逐步展开 | 破解规则、进入新区域、获得关键信息 |
| 家庭修复与责任 | 看见人物弥补旧错并承担照料责任 | 停止逃避、兑现承诺、重建信任和生活 |
| 兄弟、伙伴与传承 | 看见信任通过共事和风险累积 | 队友独当一面、导师交棒、团队彼此托底 |
| 使命与公共价值 | 看见个人成长连接更大责任 | 保护社区、完成科研、治理、救援或守护任务 |
| 自我重建 | 看见失败、破产、中年或失势之后重新行动 | 建立新能力、新事业、新关系和新判断 |

### 选择规则

- 主要承诺必须在开篇可被感知，在当前阶段完成一次实质跃迁。
- 次要承诺服务主要承诺。例如“家庭修复”可以给“事业重建”增加价值重量，但不能在中途取代事业线。
- 公开打脸只是兑现形式之一，不是主要承诺本身。
- 爱情和红颜关系均为可选；关闭爱情线时，不自动生成暧昧、后宫或女性崇拜，非爱情关系密度另行配置。

## 三、故事承诺与进展配置

### 1. 先回答十个问题

1. 主角当前与目标之间有多大差距？
2. 当前阶段可衡量的目标是什么？
3. 主角已有的真实能力和方法是什么？
4. 他缺少什么能力、资源、权限或认知？
5. 哪个问题迫使他先采取行动？
6. 他的优势有什么边界、成本或反制？
7. 哪个对手或规则能造成真实失败？
8. 每次胜利会增加哪项故事资产？
9. 新资产会带来什么新约束或责任？
10. 当前阶段结束时，主角能做什么过去做不到的事？

### 2. 选择主要进展维度

从下列维度选择一至三个作为主轴，其余作为辅助：

- **技能**：战斗、医术、工程、谈判、推理、经营、运动、创作或生存；
- **资源**：资金、装备、材料、渠道、土地、时间或补给；
- **声望**：行业信用、群众信任、排名、军功或公共评价；
- **团队**：互补成员、岗位、协作方式和组织文化；
- **权限**：决策权、执照、职位、信息访问或治理权；
- **信息**：规则、地图、证据、技术、敌人结构或历史真相；
- **组织范围**：企业、宗门、基地、城池、舰队、社区或联盟；
- **责任**：需要保护的人、必须兑现的承诺和更大公共后果。

每次阶段胜利至少更新一个主轴，并说明它如何改变下一步问题。不要让所有维度同时暴涨。

### 3. 定义阶段差距

使用可验证的差距，不只写“很弱”“很穷”“被看不起”。例如：

- 尚未获得某项资格或权限；
- 设备只能完成基础任务；
- 团队缺少关键岗位；
- 证据不足以进入下一层调查；
- 势力仅能保护一个地点；
- 技能存在明确瓶颈；
- 家庭关系仍拒绝信任他的承诺；
- 主角没有能力承担更大范围的后果。

### 4. 爱情和女性角色均为可选

- 不强制设置女主、红颜知己或后宫；
- 爱情线为 0 时，不生成女性崇拜、感情奖励或数量型关系；
- 有爱情时，双方必须各自拥有目标、能力、边界和决定权；
- 女性角色可以是主角、对手、导师、队友、领导者、家人、客户或共同责任人；
- 女性角色帮助主角时，写明她自己的利益、判断和代价；
- 主角不能以财富、权力、救命之恩或暴力能力兑换感情归属。

## 四、题材矩阵

将题材当作进展系统和问题发动机，不当作现成打脸清单。每次从“核心题材 + 一个次类型 + 一个差异化规则”开始。

### 题材校准原则

- 保留历史古代、玄幻仙侠、都市幻想、逆袭、末世、规则任务、经营和团队建设等强类型入口，但不把流行标签视为永久成功公式；
- 面向年轻受众使用奇幻脑洞、系统或能力机制时，把新鲜度落实到世界规则、反制方式、策略选择和责任升级，而不是只增加奖励数量；
- 同时保留职业、现实生活、家庭重建、社区治理、体育和技艺等跨受众题材，让男频版本能够服务不以纯打脸为目标的项目；
- 同一热门题材优先通过职业细节、组织关系、地域生活、人物价值选择和制作质量形成差异，避免换名字后重复同一羞辱—曝光—臣服结构；
- 当前市场偏好只作为需要另行调研的选题变量。用户没有要求市场研究时，按项目承诺和制作条件选择题材，不引用过时趋势替用户做决定。

| 题材族 | 创作种子 | 主要进展资产 | 主要问题发动机 | 重点风险 |
|---|---|---|---|---|
| 都市身份与人生逆袭 | 失势归来、职业重启、退婚后成长、小人物翻身、中年重启 | 技能、信用、事业、家庭关系 | 被侵占成果、资格缺口、旧债和行业权力 | 低收入职业羞辱；只靠隐藏身份 |
| 职业、创业与工业问题 | 科技、制造、物流、餐饮、法律、商业、工程、直播创业 | 方法、团队、渠道、专利、现金流、声望 | 真实业务难题、质量事故、竞争和伦理 | 把商业写成赌博；专业细节失真 |
| 医疗、民俗与奇案 | 中医题材、鉴宝、民俗、风水幻想、灵异单元案 | 专业判断、证据、信誉、规则知识 | 病案谜题、物件历史、异常规则和骗局 | 虚构疗法冒充建议；迷信现实化 |
| 历史、治理与经营 | 皇子、寒门、县令、谋士、商贾、基建、边关 | 官职、粮草、民心、制度、联盟、城市能力 | 财政、治理、战争后果、权力制衡 | 现代知识无成本碾压；权谋只靠先知 |
| 玄幻、修仙与高武 | 神脉、转世、丹阵、宗门、高武、御兽、职业觉醒 | 修为、功法、装备、门派、规则知识 | 境界瓶颈、资源稀缺、宗门秩序和天灾 | 等级随意；血脉自动胜利；敌人送经验 |
| 末世、科幻与星际 | 灾变、基地、机甲、舰队、星球经营、赛博社会 | 生存资源、技术、队伍、区域、文明信息 | 补给、环境、技术伦理、未知文明 | 科技名词堆砌；牺牲配角只抬主角 |
| 悬疑、刑侦与规则任务 | 追凶、卧底、反诈、密室、无限任务、记忆谜局 | 证据、权限、盟友、规则知识、时间 | 信息不对称、证据合法性、规则陷阱 | 犯罪教程化；警方程序失真；靠巧合破案 |
| 体育、电竞与技艺竞技 | 赛车、球类、电竞、国术、厨艺、荒野、工业竞赛 | 技术动作、体能、战术、团队默契、排名 | 训练瓶颈、比赛策略、伤病和团队冲突 | 一场顿悟无视训练；对手全靠作弊 |
| 年代创业与家庭责任 | 八九十年代、下岗重启、乡村产业、家企重建、父子修复 | 技能、渠道、家庭信任、社区资源 | 时代限制、现金流、代际误解和旧错 | 用年代情怀掩盖事实；男性单向救世 |
| 组织、势力与公共使命 | 科研、救援、城池、商会、宗门、舰队、社区治理 | 组织能力、权限、制度、人才、公共信用 | 协调、治理、资源分配和外部威胁 | 空喊家国；组织只靠个人威望 |
| 武侠、江湖与传承 | 门派、镖局、医馆、武道、师徒、江湖旧案 | 技艺、门规、信誉、盟约、传承 | 旧债、门派伦理、规则与个人自由 | 地下暴力浪漫化；武力取代所有谈判 |
| 轻喜职业与反套路 | 普通岗位满级、生活技能、社区任务、非典型英雄 | 生活能力、信任、网络、微型事业 | 日常难题、误认和规则错位 | 只靠反差梗；普通人被当作笑料 |

### 差异化规则

从以下方向选择一项，并写清如何持续制造选择：

- 可计量但有代价的能力；
- 特定行业流程或真实技术难题；
- 一支岗位互补的团队；
- 能不断扩展的地图、组织或任务网络；
- 公开排名、信用或声望系统；
- 资源稀缺与分配规则；
- 历史制度、民俗物件或案件证据链；
- 主角必须在个人收益与公共责任间反复取舍；
- 一项主角擅长、对手也能学习和反制的方法。

差异化规则必须影响至少三次关键决策，不可只作为第一集噱头。

## 五、核心人物与进展引擎

### 1. 建立进展契约

为核心主角写明：

- 当前处境与阶段差距；
- 当前阶段可验证的目标；
- 已掌握的技能和方法；
- 缺失的资源、能力或权限；
- 一项决定人物气质但不保证成功的优势；
- 一项会造成失败的盲点；
- 不会跨越的道德或责任边界；
- 当前阶段至少三次主动选择；
- 每次选择增加的资产和付出的代价；
- 阶段终点新增能力与新增责任。

### 2. 主角种子

以下种子只提供起点，必须补足进展路径和约束。

#### 都市职业与创业型

工程师、工厂技术员、厨师、律师、谈判者、物流经营者、创始人、维修专家、内容创作者、普通岗位专家。

设计重点：

- 真实问题是什么；
- 专业方法如何分步骤生效；
- 资源和现金流如何限制选择；
- 主角怎样从个人英雄转向组织能力。

#### 失势重建与身份反差型

破产者、退役者、被夺成果者、职业老将、中年父亲、失去职位者、隐退高手。

设计重点：

- 隐藏身份不能自动解决现实缺口；
- 他必须重新证明什么；
- 旧成功经验在哪里失效；
- 他是否愿意承认错误并学习新规则。

#### 历史治理与经营型

寒门书生、县令、谋士、商人、边关基层、落魄继承人、穿越工程者。

设计重点：

- 时代制度和资源约束；
- 政策或技术的副作用；
- 民心、财政、组织和安全如何互相牵制；
- 现代知识不能无成本直接复制。

#### 修炼、冒险与求生型

废柴觉醒者、转世强者、御兽者、高武学生、灾变幸存者、舰长、基地负责人。

设计重点：

- 能力规则和升级门槛；
- 对手如何反制；
- 个人强度如何转化为团队或组织责任；
- 每次升级解决什么，又引出什么。

#### 调查、规则与智斗型

侦探、刑警、反诈人员、鉴宝师、规则任务参与者、民俗调查者、情报分析者。

设计重点：

- 证据如何获得并验证；
- 主角的推理哪里可能错；
- 合法性和伦理如何约束手段；
- 真相揭示如何改变任务，而非只解释过去。

#### 体育、技艺与传承型

运动员、教练、车手、电竞老将、工匠、厨师、荒野专家、门派传人。

设计重点：

- 训练、伤病、技术动作和战术；
- 天赋不能替代积累；
- 对手拥有不同但成立的方法；
- 传承是复制旧路还是形成新路。

### 3. 进展不能只靠数字

有效进展至少符合一项：

- 主角能完成过去做不到的任务；
- 主角拥有更大的选择空间；
- 团队可以脱离主角独立执行；
- 对手被迫改变策略；
- 规则或组织结构被真实改变；
- 主角承担更大责任并建立相应能力；
- 旧关系因持续行动开始改变。

等级、财富、关注数、部下数和头衔只有改变行动能力时才有意义。

## 六、特殊优势与金手指契约

金手指、系统、血脉、神眼、重生记忆和隐秘身份均为可选。现实职业剧可以完全不用。

### 1. 七项契约

使用特殊优势前，写清：

1. **来源**：为什么是主角拥有；
2. **功能**：具体能做什么；
3. **边界**：明确不能做什么；
4. **成本**：时间、资源、身体、伦理、关系或风险成本；
5. **反制**：对手或环境如何降低其效果；
6. **升级条件**：通过什么行动而非随机奖励提升；
7. **滥用后果**：主角错误使用时会失去什么。

### 2. 使用规则

- 特殊优势提供信息、机会或方法，不直接替主角完成关键选择；
- 第一次展示说明功能，第二次增加限制，第三次让主角创造新用法；
- 奖励必须与任务因果相关，不连续凭空送财富、忠诚和权限；
- 对手应能观察、学习、隐藏信息或改变战场；
- 能力越强，责任、反制或选择难度越高；
- 隐藏身份应逐层改变局势，不用一次曝光让所有冲突消失。

### 3. 身份反差的三段释放

可使用：

1. **小露一手**：观众看到真实能力，但配角仍有合理怀疑；
2. **证据加码**：能力在更难问题中被验证，对手开始调整；
3. **公开兑现**：身份、成果或权限正式生效，改变资源和权力关系。

公开兑现后立即进入新问题。不要让群众惊讶和反派道歉占满整集。

## 七、对手层级与问题升级

### 1. 建立对手卡

为每个主要对手写明：

- 目标；
- 资源和权限；
- 已掌握的信息；
- 方法和优势；
- 对主角能造成的具体损失；
- 对主角的误判；
- 失败后会学到什么；
- 下一步策略；
- 不会跨越或最终会跨越的边界；
- 被击败后改变什么规则或格局。

### 2. 使用问题层级，不只使用人物层级

升级可以来自：

- 个人技能竞争；
- 团队协作问题；
- 资源与供应链问题；
- 组织与制度问题；
- 地域、世界或文明级规则；
- 主角责任扩大后的伦理选择。

新问题应改变解决方法。不要把“地方富二代—省城富二代—更大富二代”当作完整层级。

### 3. 让对手具备学习能力

- 第一次失败后识别主角方法；
- 第二次改变资源、规则或舆论环境；
- 第三次迫使主角放弃旧优势、依靠团队或承担代价；
- 终局对手最好在价值观或秩序方案上与主角形成真实分歧。

### 4. 打脸必须有证据和后果

一次公开反转至少包含：

- 前置质疑或不公；
- 主角可验证的行动；
- 证据、结果或第三方规则生效；
- 对手失去资源、信用、权限或策略位置；
- 主角获得新空间，同时面对新的责任或问题。

禁止只写“所有人震惊、跪下、后悔”而不改变局势。

## 八、团队、导师、家庭与关系

### 1. 团队成员必须有岗位

每名核心队友至少拥有：

- 独立技能；
- 独立目标；
- 与主角的利益连接；
- 一项会制造分歧的价值或盲点；
- 一次不等主角命令的主动选择；
- 一个能独立完成的阶段任务；
- 成长后对团队结构的真实贡献。

“忠心小弟”不能只喊大哥。忠诚应来自共同经历、价值认同、契约、互惠或被尊重。

### 2. 导师只提供方法和门槛

导师可以：

- 指出主角看不见的问题；
- 提供训练方法、历史信息或试炼入口；
- 设定伦理或能力门槛；
- 在关键时刻拒绝替主角承担责任；
- 最终交棒或接受被超越。

导师不能直接完成主角的关键任务、送出无限资源或持续掩盖主角错误。

### 3. 家庭责任是双向关系

不要默认男性价值等于独自挣钱、保护妻儿和替所有人解决问题。可以写：

- 承认过去缺席或错误；
- 学会请求和接受帮助；
- 与伴侣共同决策；
- 尊重孩子、父母或手足的选择；
- 用稳定行动而非一次财富展示重建信任；
- 在事业和照料之间建立可持续安排。

### 4. 女性角色主体性门槛

每名重要女性角色至少写明：

- 她自己的目标；
- 她拥有的专业能力或资源；
- 她帮助或反对主角的理由；
- 她不会为了主角跨越的边界；
- 她独立做出的一次选择；
- 她的选择如何改变主线；
- 即使没有爱情，她的角色弧如何成立。

禁止：

- 用美貌和身份替代能力；
- 让女性只负责被救、崇拜、吃醋、送资源或等待主角；
- 用多名女性争夺主角证明主角成功；
- 把女性数量作为奖励或升级参数；
- 因主角救过她就自动兑换爱情；
- 让女队友的错误只为衬托男主永远正确。

### 5. 爱情线规则

- 爱情权重可设为 0；
- 双方各自的事业、责任和判断必须存在；
- 关系推进通过共事、验证、沟通和承担；
- 不用财富、职位、救命之恩或暴力实力购买感情；
- 关系冲突不能长期依赖拒绝正常沟通；
- 不复合、不结婚或暂时分开都可以是成立的结局。

## 九、成长弧与兑现谱系

### 1. 可复用的成长弧

#### 能力掌握弧

初始方法失效 → 识别具体缺口 → 训练或实验 → 第一次应用失败 → 改进方法 → 在高压任务中掌握 → 将方法传给团队。

#### 事业建设弧

个人技能谋生 → 获得第一项稳定资源 → 找到互补伙伴 → 建立流程 → 遭遇组织级危机 → 调整制度 → 形成可持续事业。

#### 失势重建弧

失去位置或信用 → 承认旧方法的问题 → 从小任务重新证明 → 兑现第一份承诺 → 取回选择权 → 决定是否回到旧位置或建立新路。

#### 规则破局弧

被旧规则压制 → 理解规则真实运行方式 → 利用合法入口获得空间 → 看见规则伤害 → 联合他人提出替代方案 → 承担改革代价。

#### 团队领导弧

独自解决 → 因个人极限失败 → 招募互补成员 → 学会授权 → 面对价值分歧 → 建立共同规则 → 团队能独立行动。

#### 家庭责任弧

逃避或只会物质补偿 → 旧伤再次暴露 → 学会倾听和承认 → 持续完成小承诺 → 面对事业与家庭冲突 → 做出可持续安排。

#### 探索与使命弧

发现异常 → 建立规则假设 → 第一次误判 → 解锁更大世界 → 个人收益与公共责任冲突 → 选择使命并承担不可逆代价。

### 2. 兑现谱系

在当前阶段选择三至五种，避免每集都靠公开羞辱：

- 解决一个真实专业问题；
- 掌握一项新技能或方法；
- 获得一项能改变行动范围的资源；
- 对手被迫改变策略；
- 队友独立完成任务；
- 团队建立新流程或共同规则；
- 主角承认错误并修正方法；
- 家庭或伙伴开始重新信任；
- 组织、社区或制度得到改善；
- 真相打开更大任务；
- 主角放弃短期利益承担责任；
- 主角将个人能力转化为他人也能使用的系统；
- 公开身份或成果生效，但同时带来新义务。

### 3. 热血来自难度与选择

需要高强度爽感时：

- 先让目标和差距可见；
- 让对手或问题拥有真实优势；
- 展示主角判断和执行过程；
- 让胜利付出代价或暴露新风险；
- 让结果增加能力、资源、信任或责任。

不要用连续嘲讽、群众吹捧、无成本碾压或暴力升级替代热血。

## 十、男频调节参数

将参数写入项目配置和最终交付。参数可以使用 0 至 100，也可以使用低、中、高，但必须解释叙事后果。

| 参数 | 低值 | 中值 | 高值 |
|---|---|---|---|
| 进展速度 | 慢热训练与积累 | 阶梯成长 | 高频跃迁但仍保留门槛 |
| 公开兑现频率 | 重蓄力、少公开场 | 阶段性兑现 | 高频验证但必须改变局势 |
| 特殊优势强度 | 弱辅助或无金手指 | 强但受限 | 碾压局部问题、承担高约束 |
| 团队规模 | 独行或双人 | 核心小队 | 多岗位组织 |
| 势力与事业范围 | 个人与家庭 | 企业、门派或地区 | 跨区域组织或文明任务 |
| 动作戏权重 | 智斗、经营、职业为主 | 文武均衡 | 动作主导但因果完整 |
| 任务或副本数量 | 单一主线 | 少量阶段任务 | 多单元、多地图 |
| 对手层级复杂度 | 单层核心对手 | 个人—组织—终局 | 多势力、多规则竞争 |
| 爱情线权重 | 0 可完全无爱情 | 爱情为副线 | 爱情为重要承诺 |
| 非爱情关系密度 | 独行或少量功能关系 | 家庭、师徒或小队关系 | 多岗位团队、组织与多代际关系 |
| 责任重量 | 个人目标 | 家庭与团队 | 社区、组织或公共使命 |

### 参数冲突处理

- 爱情线为 0 时，删除红颜、暧昧和感情奖励，但保留项目所需的家庭、师徒、友谊与团队关系；
- 特殊优势高、进展速度高时，提高反制、代价和责任，避免自动解题；
- 独狼路线与高团队规模冲突时，明确何时从个人转向组织；
- 高频公开兑现不能重复同一种群众震惊；
- 多副本必须服务一条主进展线，并持续更新状态账；
- 动作权重低时，用专业难题、谈判、经营、推理和组织选择制造强度；
- 高家国责任不能只增加口号，必须增加具体后果和决策难度。

交付前主动清除以下问题：

- 用隐藏身份一次性解决全部矛盾；
- 让反派排队送经验、降智嘲讽或在证据面前拒绝现实；
- 让系统连续随机送财富、忠诚、权力和异性关系；

## 十二、男频硬门槛

本门槛与共同内核的硬门槛同时执行。任一适用项为 REVISE 时，先修稿再交付。

1. **阶段差距**：主角的初始差距和当前阶段目标可验证，不只是“受人轻视”。
2. **开篇行动**：开篇阶段能看见主角主动处理一个问题，而非只等待身份曝光。
3. **进展资产**：每次阶段胜利至少更新一项技能、资源、声望、团队、权限、信息、组织范围或责任。
4. **不归零**：已经获得的有效进展不会无理由消失；失去时有明确因果。
5. **优势边界**：特殊能力、身份或贵人有边界、成本、反制或责任，不能一键解题。
6. **对手有效**：主要对手拥有目标、资源、策略和学习能力，不只负责嘲讽与被打脸。
7. **兑现有后果**：公开反转包含行动和证据，并真实改变资源、信用、权限或局势。
8. **团队主体性**：重要队友拥有独立技能、目标和至少一次不等主角命令的选择。
9. **导师不代打**：导师提供方法、门槛或信息，不替主角完成关键任务。
10. **女性主体性**：重要女性角色拥有独立目标、能力、边界和选择；她的角色弧在没有爱情时仍成立。
11. **爱情可选**：爱情线权重可为 0；关闭时不生成红颜数量、女性崇拜或感情奖励，且不误删重要的非爱情关系。
12. **责任可见**：家庭、团队、社区或家国责任通过具体风险、代价和行动体现，不靠口号。
13. **无权力崇拜**：财富、暴力、职位、地下势力和臣服没有被当作人物价值的唯一证明。
14. **专业可信**：职业、历史、医疗、金融、军事、刑侦和工程内容可信或明确虚构，不构成现实建议。
15. **无性别定律**：没有把男频规则写成对所有男性的心理结论。
16. **本季跃迁**：开启分季时，本季完成一次能力、事业、组织、关系或使命的实质跃迁，并引出新的约束而非重复冲突。
