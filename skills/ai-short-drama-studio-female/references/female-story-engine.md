# 女频故事引擎

本文件在 [编剧共同内核](screenwriting-core.md) 之上提供女频版的选题、人物、关系、情绪兑现和质量门槛。先执行共同内核，再执行本文件。

“女频”是发行与叙事惯例的市场分类，不是对所有女性的心理诊断。不要假设女性天然只需要爱情、帅哥、豪门或被拯救。根据具体项目选择观看承诺，并允许爱情线权重为 0。

本引擎提供创作判断，不承诺爆款、播放量、付费率或平台推荐。

本文的“当前阶段”在开启分季时指当前季，在未分季时指全剧或用户指定的当前规划范围。

## 一、女频分支的执行原则

1. **先确定人，再确定关系奖励。** 先写主角想完成的事、能做的事和不愿再承受的事，再决定是否需要爱情。
2. **把主体性写成行动。** “独立”“清醒”“大女主”不是标签。让主角收集信息、做选择、承担代价并改变局势。
3. **同时保护逻辑与情绪。** 情绪由动机、行动、后果和表演承载，不能以“女频重情绪”为理由降低因果标准。
4. **把偏爱翻译成可验证行为。** 信任、尊重边界、共同承担、公开负责和关键时刻的实际选择，比财富、身份和占有欲更能建立关系价值。
5. **给女性配角完整人生。** 女性角色可以竞争、犯错或作恶，但不能因为“另一个女人”这一身份自动成为恶人。
6. **让结局回应主角选择。** 圆满不等于必须结婚，悲剧不等于必须惩罚主角，开放式也不能逃避已经承诺的阶段兑现。

## 二、观看承诺系统

每个项目选择一个主要承诺和至多两个次要承诺。不要把全部承诺堆在一部作品中。

| 承诺 | 观众获得什么 | 可见兑现 |
|---|---|---|
| 主体性与选择权 | 看见人物从被安排到能决定自己的生活 | 拒绝不合理交换、主动转向、建立新规则 |
| 能力与自我实现 | 看见专业、创造和领导力产生真实结果 | 完成作品、解决问题、赢得权限、建立事业 |
| 尊重与公平 | 看见被忽视或被侵占的价值获得纠正 | 证据生效、功劳归位、规则修正、责任落实 |
| 关系安全与互惠 | 看见亲密关系中的理解、边界和共同承担 | 真实沟通、停止控制、承担代价、双向选择 |
| 恢复与重建 | 看见人物从创伤、失败或耗竭中重建生活 | 恢复判断、建立支持网络、重新行动 |
| 归属与女性协作 | 看见姐妹、同事、家人或社区形成可靠连接 | 互相托底、共同决策、资源互换、冲突后修复 |
| 正义与问责 | 看见伤害者被追责、受害者取回行动空间 | 真相公开、利益返还、制度问责、离开伤害 |
| 探索与命运选择 | 看见人物理解陌生规则并拒绝被预设命运定义 | 识破规则、改写契约、承担新使命 |

### 选择规则

- 主要承诺必须在开篇可被感知，在当前阶段获得实质兑现。
- 次要承诺只能增强主要承诺，不能让作品中途换题。
- 纯爱情剧也要给主角关系内目标，例如建立信任、确认边界或决定去留。
- 无爱情剧可以用事业、探案、家庭、社区、成长、冒险或使命承担全部动力。

## 三、故事承诺配置

### 1. 先回答八个问题

1. 主角当前被什么现实关系、制度、身份或内在信念限制？
2. 她真正想完成什么，不只是想摆脱什么？
3. 她已经拥有什么能力、资源或关系？
4. 她的能力边界和盲点是什么？
5. 哪一次选择能证明她开始掌握自己的人生？
6. 当前阶段结束时，她应获得什么真实的新空间？
7. 如果有爱情，爱情如何支持而不是取消她的判断？
8. 如果没有爱情，哪组关系承担情感连接和主题回声？

### 2. 定义非爱情目标

默认要求核心主角拥有一个非爱情目标。可选类型包括：

- 完成职业、作品、考试、案件或创业目标；
- 夺回被侵占的成果、身份、财产或名誉；
- 保护家人、社区、技艺、土地或公共利益；
- 查清旧案、身世、制度问题或超自然规则；
- 修复身体、心理、家庭或生活秩序；
- 获得选择职业、关系、居所或使命的权利；
- 建立组织、团队、事业或新的生活方式。

用户明确选择纯关系剧时，将“建立怎样的关系、拒绝怎样的关系、愿意承担什么”写成可行动目标，不能只写“得到某个人”。

### 3. 爱情和男主均为可选

- 不强制设置男主、男二或异性恋关系；
- 有爱情时，明确爱情线权重和主次位置；
- 爱情线为 0 时，不生成暧昧、追妻或强行配对；
- 双主角关系必须让双方都拥有目标、资源、盲点和选择；
- 伴侣不能替主角完成最终决定，也不能用钱权购买同意。

## 四、题材矩阵

将题材当作故事发动机，不当作现成剧情。每次从“核心题材 + 一个次类型 + 一个差异化机制”开始，避免标签堆砌。

| 题材族 | 创作种子 | 主要发动机 | 常见兑现 | 重点风险 |
|---|---|---|---|---|
| 都市关系与亲密协商 | 先婚后爱、久别重逢、成年人关系、破镜重圆、非典型家庭 | 两个人的目标和边界不断协商 | 建立信任、停止控制、共同承担、决定去留 | 把占有欲包装成爱；用财富替代沟通 |
| 职业、创业与行业成长 | 医疗、法律、设计、科技、非遗、娱乐、体育、餐饮 | 专业难题、行业权力和资源竞争 | 能力被验证、成果归位、事业成形 | 只靠头衔吹捧；专业常识错误 |
| 身份重建与正义问责 | 重生、真假身份、离婚重启、被冒名、家产争夺 | 取回身份、证据、资源和叙事权 | 真相生效、责任落实、生活重建 | 长期受辱只为一次打脸；反派全员降智 |
| 古代家族、权力与家国 | 宅院、女官、女将、联姻、探案、经商、治理 | 身份制度、家族利益和公共责任冲突 | 获得决策权、完成治理、守住使命 | 用现代口号代替时代逻辑；爱情覆盖家国线 |
| 穿越、重生与经营建设 | 种田、荒年求生、年代创业、药膳、手艺传承 | 知识转化、资源经营和社区协作 | 生产改善、组织建立、生活选择扩大 | 技术无成本开挂；把贫穷当笑料 |
| 校园、竞技与成年礼 | 暗恋、竞赛、艺术、电竞、社团、重回校园 | 身份形成、能力训练和同伴关系 | 完成比赛、表达选择、离开旧评价 | 成年权力关系侵入未成年恋爱；校园霸凌浪漫化 |
| 仙侠、玄幻与命运选择 | 师徒、轮回、双强、规则、契约、修行 | 力量规则与个人选择发生冲突 | 改写契约、承担使命、共同成长 | 权力差被包装成爱情；规则为反转随意更改 |
| 灵异、悬疑与民俗探案 | 玄学、单元案、规则怪谈、风水、时间循环 | 发现规则、收集证据、识别叙述盲区 | 真相揭示、规则破解、旧债问责 | 把民俗写成现实迷信建议；只用惊吓不推进 |
| 娱乐、舆论与公共身份 | 顶流、演员、主播、创作者、公共危机 | 私人身份与公共叙事争夺 | 作品说话、舆论反转、建立表达权 | 网络暴力被轻描淡写；以热搜代替因果 |
| 年代、家庭与社区重启 | 小城、随军、代际、银发、中年、家庭企业 | 照料责任、代际价值和社区资源重组 | 关系修复、生活重启、技艺或事业传承 | 让女性无限牺牲；用和解免除伤害责任 |
| 女性群像与共同任务 | 合伙创业、案件小组、家庭姐妹、社区联盟 | 多个目标通过合作、冲突和重新结盟推进 | 形成共同规则、共享资源、彼此成长 | 所有人只围着一个主角转；群像没有独立弧 |
| 科幻、末世与异世界 | 星际职业、灾变求生、AI社会、时间结构 | 新规则下的生存、伦理和领导选择 | 建立团队、理解规则、改变秩序 | 用宏大设定掩盖人物；技术规则不一致 |

### 差异化机制

从以下方向选择一项，并写清如何持续作用于因果：

- 特定职业流程或手艺；
- 能产生证据链的道具；
- 独特空间，如老宅、医院、剧团、县城、舰队或社区；
- 有成本的能力或规则；
- 非线性时间、双视角或不可靠叙述；
- 一组价值观不同但必须合作的人；
- 贯穿全剧的声音、气味、物件或仪式。

差异化机制必须至少影响三次关键选择，不能只作为宣传概念出现一次。

## 五、核心人物引擎

### 1. 建立主体性契约

为核心主角写明：

- 她想完成的正向目标；
- 她拒绝继续接受的条件；
- 她能主动调动的能力和资源；
- 她必须学习但尚未掌握的能力；
- 她最可能做错的判断；
- 她在压力下仍不会跨越的边界；
- 当前阶段至少三次关键选择；
- 阶段终点她获得或放弃的真实权利。

### 2. 人物种子

以下种子只提供起点，必须补足能力边界和具体行动。

#### 事业与领导型

创始人、医生、律师、设计师、演员、研究者、工匠、管理者、教练、记者、女官、商人。

设计重点：

- 专业能力解决什么具体问题；
- 她如何面对资源不足和行业权力；
- 她在效率、伦理和关系之间如何取舍；
- 她的领导方式如何改变。

#### 重启与取回型

被冒名者、离婚重启者、重生者、失去事业后返场者、被家族剥夺者、穿书改命者。

设计重点：

- 她不是只逃离旧生活，还要建立什么；
- 旧创伤如何影响判断；
- 她通过什么证据、能力或联盟取回空间；
- 取回之后是否复制旧权力模式。

#### 温柔与照料型

医生、教师、店主、照护者、社区组织者、心理工作者、宠物救助者、家庭支柱。

设计重点：

- 温柔是主动价值，不等于无法拒绝；
- 照料劳动的成本由谁承担；
- 她何时停止自我耗竭；
- 她如何接受别人支持。

#### 行动与专业冒险型

调查者、女警、女将、运动员、救援者、鉴宝师、工程师、玄学题材角色、科幻驾驶员。

设计重点：

- 技能如何训练和受限；
- 失败会造成什么现实后果；
- 她的判断如何影响团队；
- 行动力之外的盲点是什么。

#### 轻盈与成长型

学生、实习生、创作者、主播、小店主、年轻继承人、社团成员。

设计重点：

- 可爱、幽默或柔软之外，她具体想做什么；
- 她如何从依赖评价转向形成判断；
- 她的失误如何推动成长；
- 不让成熟角色替她完成全部决定。

### 3. 成长必须不可伪装

不要用换装、身份曝光、财富到账或男主认可代替成长。成长至少包含：

- 对自己处境的新认识；
- 与旧行为不同的一次选择；
- 选择带来的真实代价；
- 新行为在后续场景中被再次执行；
- 关系或生活结构因此改变。

人物可以反复、迟疑或暂时退步，但必须能看见反复的原因和重新选择。

## 六、关系架构

### 1. 可选架构

| 架构 | 核心问题 | 写作要求 |
|---|---|---|
| 无爱情主线 | 主角如何建立自我、事业、家庭或共同体 | 用伙伴、家人、师徒或社区承担情感连接 |
| 单伴侣线 | 两个人能否建立可持续的互惠关系 | 双方都拥有目标、边界和改变任务 |
| 势均力敌双主角 | 两种能力或价值如何冲突并协作 | 不让一方只负责被拯救或提供资源 |
| 慢热信任线 | 人物如何从谨慎合作走向真实信任 | 信任通过多次可验证行动累积 |
| 女性同盟 | 不同女性如何在分歧中建立合作 | 保留立场差异，不写成无条件姐妹口号 |
| 家庭与代际 | 爱、责任、控制和旧伤如何重新划界 | 和解必须包含承认、赔偿或行为改变 |
| 师徒与传承 | 知识、权力和价值如何交接 | 导师不垄断答案，后辈能质疑并超越 |
| 群像共同任务 | 多个目标如何围绕共同危机重组 | 每名核心角色拥有独立任务和选择 |

### 2. 亲密关系的可验证标准

一段关系被写为正向关系前，至少能看见：

- 尊重拒绝和知情同意；
- 不用监控、羞辱、威胁、封锁资源或职位施压；
- 在信息不足时先保护基本尊严，再共同核实事实；
- 能承认错误并承担可见后果；
- 不要求对方放弃全部事业、朋友或判断；
- 冲突后出现新的行为，而非只说“我爱你”。

### 3. 伤害与修复

“追妻火葬场”统一转换为责任与修复设计：

1. 伤害是什么；
2. 对主角造成了什么实际后果；
3. 伤害者是否完整承认；
4. 需要停止什么行为；
5. 需要偿还或修复什么；
6. 主角是否愿意重新建立关系；
7. 即使不复合，双方的弧是否仍成立。

原谅不是必选结局，公开道歉也不能自动兑换亲密关系。

## 七、配角与对抗设计

### 1. 从利益与价值产生反派

优先使用：

- 对成果、继承、职位、证据或资源的争夺；
- 对规则、家庭责任、职业伦理或公共利益的不同理解；
- 旧罪、旧债和身份秘密；
- 机构、行业或家族中的激励结构；
- 对失去权力、名誉、关系或归属的恐惧；
- 与主角同样合理但不可兼容的目标。

不要把“婆婆、闺蜜、情敌、前任、嫡母、女同事”直接当作反派动机。

### 2. 给反派有效能力

写明反派：

- 想保住或得到什么；
- 掌握什么资源；
- 能对主角造成何种真实损失；
- 如何根据失败调整策略；
- 哪条边界会跨越，哪条不会；
- 被问责时失去什么。

反派可以令人厌恶，但不能只会站着辱骂。主角的胜利必须克服反派真实优势。

### 3. 建立非竞争型女性角色

如果作品有两名以上重要女性角色，至少一名拥有不依赖争夺同一男性的目标。可以设置：

- 互补专业；
- 价值观分歧；
- 资源交换；
- 曾经伤害后的修复；
- 竞争后形成尊重；
- 共同面对更大制度问题；
- 选择不同人生而仍保持连接。

## 八、成长弧与兑现谱系

### 1. 可复用的成长弧

#### 清醒与离场

识别伤害或不平等 → 尝试旧方法 → 看见旧方法无效 → 建立边界 → 承担离场代价 → 建立新生活。

#### 取回与重建

失去身份、成果或资源 → 找到证据和能力入口 → 获得第一项可控资源 → 形成联盟 → 公开取回 → 决定如何使用新权力。

#### 专业与领导

拥有天赋但缺乏位置 → 经历真实失败 → 完成训练或方法升级 → 解决关键问题 → 获得团队信任 → 建立自己的规则。

#### 关系与互惠

互相误读或目标冲突 → 被迫合作 → 通过行动验证 → 暴露真实需求 → 面对一次边界考验 → 双向选择或体面分开。

#### 恢复与归属

孤立、耗竭或创伤后停滞 → 出现安全连接 → 尝试小行动 → 允许别人支持 → 面对复发或旧环境 → 建立可持续生活。

#### 调查与使命

异常出现 → 建立规则假设 → 第一次判断失败 → 发现制度或历史根因 → 在个人利益与公共责任间选择 → 完成阶段使命。

### 2. 兑现谱系

在当前阶段选择三至五种，避免只用“公开打脸”：

- 主角做出过去不敢做的选择；
- 专业能力解决具体问题；
- 功劳、身份、作品或财产得到有效确认；
- 主角停止解释并转向行动；
- 一段关系建立新边界；
- 伤害者承担责任；
- 女性角色形成可靠合作；
- 一个家庭或社区规则被改变；
- 主角允许自己接受帮助；
- 旧信念被新行动替代；
- 亲密关系完成双向承诺；
- 主角选择不复合、不继承或不接受看似诱人的权力。

### 3. 情绪强度来自落差和代价

需要高强度情绪时：

- 先建立人物在乎什么；
- 让威胁具体触及这一价值；
- 给人物可选但困难的行动；
- 让行动付出代价；
- 让结果改变关系或局势。

不要用连续羞辱、重复误会、突发绝症或无尽受苦充当情绪深度。

## 九、女频调节参数

| 参数 | 低值 | 中值 | 高值 |

### 参数冲突处理

- 爱情线为 0 时，删除“男主动心速度、追妻、误会恋爱”等派生要求，但保留项目所需的同事、家庭、师徒、友谊与女性协作；
- 高甜与严重伤害同时出现时，先指出冲突，再选择“降低伤害”或“把甜放到另一段健康关系”；
- 快速成长与慢热长剧同时出现时，把“早期选择变化”和“长期能力深化”分开；
- 高信息差不能通过全员拒绝沟通维持；
- 高群像必须减少每个人的标签数量，增加独立任务和交汇点。

## 十、陈旧套路与敏感边界

交付前主动清除以下问题：

- 把女性受众概括为缺男人、缺宠爱或只想嫁入豪门；
- 强制设置男主、男二、三角恋或异性恋结局；
- 用男主的钱、权、武力代替女主解决最终问题；
- 把监控、限制社交、强吻、职位胁迫、资源封锁写成浪漫；
- 用怀孕、流产、疾病、失忆、残障或容貌变化惩罚女性；
- 用贫穷、普通职业、离婚、中年或未婚制造价值羞耻；
- 让温柔角色无限牺牲，让强势角色只靠羞辱别人证明力量；
- 把所有女性配角写成嫉妒、拜金、抢男人或无条件助攻；
- 用“全员偏爱女主”替代女主行动；

- 把民俗、医疗、法律或投资桥段写成现实建议。

## 十一、女频硬门槛

本门槛与共同内核的硬门槛同时执行。任一适用项为 REVISE 时，先修稿再交付。

1. **非爱情目标**：核心主角拥有明确的非爱情目标；纯关系剧则拥有可行动的关系目标。
2. **开篇主体性**：开篇阶段能看见主角至少一次主动判断、选择或拒绝。
3. **决定性行动**：当前集或当前阶段至少一个关键进展来自主角自身行动，不由伴侣代办。
4. **能力证据**：主角的专业、智慧或照料能力通过具体方法和结果展示，不靠旁人吹捧。
5. **爱情可选**：爱情线权重可为 0；关闭时没有自动生成的暧昧、追妻或配对，且不误删重要的非爱情关系。
6. **关系边界**：正向亲密关系尊重同意、边界和独立目标；伤害有后果，修复有成本。
7. **配角主体性**：重要女性配角至少拥有一个不依赖争夺伴侣的目标和一次独立选择。
8. **反派动机**：反派由利益、价值、恐惧或旧债驱动，不因性别或家庭身份天然邪恶。
9. **成长有效**：成长包含认识、选择、代价和后续重复行为，不用换装、身份或爱情代替。
10. **兑现多样**：当前阶段不只依赖羞辱后打脸，至少包含一种能力、边界、创造、协作或修复兑现。
11. **结局一致**：结局回应主角的价值选择，不为圆满强迫复合、结婚或放弃已经建立的生活。
12. **无伤害美化**：控制、羞辱、暴力和权力胁迫没有被包装为深情。
13. **无性别定律**：没有把女频规则写成对所有女性的心理结论。
14. **本季闭环**：开启分季时，本季完成一个主体性、事业、关系、正义或归属的实质承诺。
