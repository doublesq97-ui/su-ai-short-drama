# su-ai-short-drama

`su-ai-short-drama` 是一套面向中文短剧创作与 AI 辅助制作的开源工作流，由两个可独立安装的 Codex Skill 组成：

- `ai-short-drama-studio-female`：女频创作配置。
- `ai-short-drama-studio-male`：男频创作配置。

两版共享同一套专业生产逻辑，但在受众假设、题材库、人物动力、情绪回报和节奏调节上分别设计。你可以只安装其中一版，也可以同时安装。

> 维护状态：**Low maintenance / best effort**。本项目按现状提供，不承诺持续更新、问题响应、模型兼容、生成效果、平台审核结果或商业表现。

## 9 秒看懂工作流

[![su-ai-short-drama 工作流演示](docs/media/su-ai-short-drama-workflow.gif)](docs/media/su-ai-short-drama-workflow.mp4)

创意输入 → 选择女频版或男频版 → 选择当前岗位 → 共同读写项目档案 → 输出剧本、角色卡、场景卡和分镜交付。点击动图可查看 1080p MP4。

## 两个 Skill 的边界

| Skill | 主要创作重心 | 常见题材与体验 |
|---|---|---|
| 女频版 | 主体成长、关系变化、情绪张力、偏爱与信任、复仇或事业回报 | 都市情感、甜宠、逆袭、重生、宅斗、校园、仙侠、悬疑情感 |
| 男频版 | 能力成长、身份兑现、秩序重建、事业扩张、策略与行动回报 | 都市逆袭、职业成长、权谋经营、修仙玄幻、战神神医、末世、高武、悬疑冒险 |

“女频 / 男频”在这里是内容市场中的创作配置，不是对所有女性或男性的心理判断。两版都可以包含不同性别的主角、复杂人物关系和跨类型故事，也不要求作品必须落入固定套路。

同一项目通常选择一版作为主 Skill，避免两套节奏规则互相冲突。需要做跨受众比较或混合类型时，可以明确要求 Codex 分别提出两版方案，再由人决定最终方向。

## 剧本生成器

本套装同时包含两个独立的可视化剧本生成器，不依赖网络，双击 HTML 即可打开：

- [女频版剧本生成器](tools/script-generators/女频版剧本生成器.html)：言情、甜宠、重生、宅斗、事业逆袭等配置。
- [男频版剧本生成器](tools/script-generators/男频版剧本生成器.html)：逆袭、打脸、修仙、战神、神医、权谋等配置。

两版都支持长剧 / 短剧、分季开关、总季数、单季集数和对应的可视化调节参数。它们用于整理创作参数，不会自动替代人工审核，也不会修改仓库中的 Skill 规则。

## 岗位模式

两个 Skill 都支持单独调用某一岗位，也支持沿完整制作链协作：

- **总导演 / Showrunner**：立项、任务路由、创作口径、质量门槛、工种交接和最终一致性。
- **编剧**：选题、人物弧、世界观、分集大纲、单集剧本、对白和改稿。
- **选角与角色设计**：选角简报、虚构角色视觉设计、定妆、阶段造型及人工审批。真实演员选角与 AI 角色生成必须区分处理。
- **摄影与分镜导演**：场面调度、景别、机位、运镜、分镜和可执行的视频生成提示。
- **美术与场景导演**：空间结构、场景卡、色彩、光线、道具和视觉连续性。
- **表演导演**：角色行为、表情、动作链、对白意图和反待机感。
- **声音与后期协调**：对白、旁白、环境声、音乐、音效、字幕和剪辑交接。
- **场记与连续性管理**：项目档案、命名、版本、资产状态、剧情连续性、来源与授权记录。

你不需要一次启动全部岗位。比如只让编剧完成剧本，之后再将已确认的剧本交给总导演或摄影分镜岗位即可。

## 工作方式

建议将每个项目的已确认信息放进独立的项目档案，并把它作为工种之间的共同事实来源：

1. 明确版本、题材、体量、分季方式和交付目标。
2. 选择主 Skill 和当前岗位。
3. 建立人物、场景、剧情与资产档案。
4. 先产出候选方案，再由人审核和锁定。
5. 锁定内容进入下一工种；未锁定内容不得冒充正式资产。
6. 每次交接更新当前进度、决定记录、未解决问题和下一步。

模板中的集数、镜头时长、对白速度、每批交付数量和叙事比例都是可调整参数，不是适用于所有模型和项目的固定定律。

## 适用场景

- 中文竖屏短剧、分季短剧和连续剧式短内容。
- 从模糊创意到人物、剧本、分镜和制作交接。
- 编剧与导演分开工作，或由总导演协调多个岗位。
- AI 图像、视频和声音工具参与，但保留人工审批的制作流程。
- 需要长期维护角色、场景、剧情和资产连续性的项目。
